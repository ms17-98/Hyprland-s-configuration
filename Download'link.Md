# 获取地址
Hyprland配置获取地址：
https://wwsy.lanzoub.com/b00hrcbbhc
提取码:as28
下载到Hyprland-Dots.7z后
用ark工具解压
或者就是7z工具解压
获取config文件夹之后
创建一个目录
如 mkdir -p “你的用户目录”
如：/home/Administrator/Arch-Hyprland-Dots
cd进入你创建的的Arch-Hyprland-Dots目录
执行sudo cp -r config "你创建的Arch-Hyprland-Dots文件夹路径"
# 补充一点 在进入目录前需要在我的github上下载我的install.sh脚本拷贝到同一个目录必须是同一个目录
# 命令如下
chmod +x install.sh
bash install.sh
脚本跑完之后
执行sudo mv config .config
再执行 mv .config "到你的用户主目录"
注销当前桌面进入用户登陆界面切换到Hyprland桌面就行了
